# 用户操作流程图

## 1. 登录/注册流程
```
开始 -> 访问应用 -> 显示登录页面 -> 输入凭证 -> 验证 -> 登录成功 -> 跳转至仪表盘
                                         |
                                         v
                                      验证失败 -> 显示错误信息 -> 重新输入

如果没有账号 -> 点击注册 -> 显示注册页面 -> 填写信息 -> 提交 -> 验证 -> 注册成功 -> 发送确认邮件 -> 跳转至登录页面
                                                                 |
                                                                 v
                                                              验证失败 -> 显示错误信息 -> 重新填写
```

## 2. 任务管理流程
```
开始 -> 访问仪表盘 -> 查看任务列表 -> 选择操作:
  |-> 创建任务 -> 填写任务信息 -> 保存 -> 任务添加成功 -> 更新任务列表
  |-> 编辑任务 -> 修改任务信息 -> 保存 -> 任务更新成功 -> 更新任务列表
  |-> 删除任务 -> 确认删除 -> 任务删除成功 -> 更新任务列表
  |-> 标记完成 -> 任务状态更新 -> 更新任务列表
```

## 3. 时间追踪流程
```
开始 -> 访问任务详情 -> 开始计时 -> 任务进行中 -> 暂停计时 -> 记录耗时 -> 更新任务时间统计
                                 |
                                 v
                              完成任务 -> 停止计时 -> 记录总耗时 -> 更新任务状态 -> 生成时间报告
```

## 4. 报表生成流程
```
开始 -> 访问报表页面 -> 选择时间范围 -> 选择报表类型 -> 生成报表 -> 查看报表 -> 选择操作:
  |-> 导出PDF -> 下载PDF文件
  |-> 分享报表 -> 选择分享方式 -> 发送
  |-> 重新生成 -> 选择新参数 -> 生成新报表
```
